## 仓库用途

本仓库为 [Anima](https://github.com/Origin-of-Miracles/Anima) 模组提供：

- **人格配置**（Persona Configs）：定义学生的 AI 对话行为
- **GeckoLib 模型**：3D 模型、动画、纹理文件

将资源与代码分离，让社区更便捷地贡献内容，同时保持主仓库轻量化

---

## 目录结构

```
Anima-Assets/
├── personas/                    # 人格配置文件
├── models/
│   ├── geo/students/           # GeckoLib 几何模型
│   ├── animations/students/    # 动画定义
│   └── textures/students/      # 纹理文件
└── scripts/
    └── validate.py             # 资源验证工具
```
